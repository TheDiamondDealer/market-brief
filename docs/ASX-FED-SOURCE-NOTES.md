# ASX and Federal Reserve Source Notes

This integration is limited to public official metadata and links.

- ASX collection uses the public per-company announcement endpoint for the configured watchlist. It is not the licensed complete real-time ComNews service and must not be described as market-wide or redistribution-enabled.
- Federal Reserve collection uses the Board's official RSS feeds for monetary-policy releases, speeches, testimony, and credit/liquidity or balance-sheet releases.
- Both sources retain the last verified snapshot as stale on temporary failure.
- Neither source automatically creates a directional market interpretation or trading signal.
- USTR, US Treasury and Commerce/BIS release monitoring remain separate future extensions until stable official machine-readable endpoints and source-specific parsers are verified.
