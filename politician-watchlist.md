# Political Disclosure Watchlist

The dashboard tracks delayed public financial disclosures and estimated disclosure-derived portfolios for:

- Donald Trump — policy, tariff actions and public financial disclosures
- Nancy Pelosi
- Tim Moore
- Dan Meuser
- Cleo Fields
- Rob Bresnahan
- Donald Beyer
- Sheldon Whitehouse
- Josh Gottheimer

## Update rules

- Check the relevant official House, Senate or executive-branch disclosure source before third-party parsers.
- Retain every verified transaction permanently, including filings reported 45 or more days after execution.
- Use the actual transaction date for chronology and keep the filing date and disclosure lag separate.
- Keep member, spouse, joint, dependent, trust and managed-account ownership exactly as disclosed.
- Reconstruct each estimated portfolio from the latest verified annual holdings disclosure plus every later verified purchase, sale, exchange and option exercise.
- Do not import third-party backtest returns or strategy performance as if they were official portfolio performance.
- Do not infer an exact holding value from a statutory range.
- Add future names to `site/politicians.js` and to this watchlist.
